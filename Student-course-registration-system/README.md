# Student Course Registration & Result Management System

This project is a Java-based system developed as part of the Winter Workshop mini project.

## Overview
The system manages:
- Student registration
- Course enrollment
- Grade assignment
- GPA and CGPA calculation
- Transcript generation

It is designed to demonstrate core Object-Oriented Programming concepts including abstraction, inheritance, polymorphism, interfaces, exception handling, collections, and file handling with data persistence.

A simple frontend interface is included to improve usability and presentation.

## Technologies Used
- Java
- File Handling & Object Serialization
- HTML, CSS, JavaScript (Frontend)
- Git & GitHub
